k8s可觀測性實戰課程資源
=========================

此倉庫是放置課程資源，既使是沒有授課的也歡迎大家取用，當然還是希望大家能幫忙支持一下我的課程:)。  
課程主要分為四個段落以及多個章節。有基本知識篇，可觀測性篇，分布式可觀測性篇，實際案例。對於已經有在使用k8s的使用者來說，如何以cloud native的方式來收集程式紀錄，部署程式狀態以及當錯誤發生如何追蹤等等。真心覺得是一們簡單易上手的k8s監控課程。值得擁有。
